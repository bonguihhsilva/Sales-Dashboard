# Quick Task 260701-rm0: Adicionar trilha Perfumaria (13 modulos) ao LMS

## Objetivo
Adicionar uma nova trilha global de treinamento "Perfumaria de Alto Nível" ao LMS do dashboard, com 13 módulos de conteúdo (história, anatomia, famílias olfativas, ingredientes, casas, árabe, nicho, técnicas de venda, mercado CDE, inteligência comercial, simulações), cada um com aula + prova de 5 questões.

## Tarefas
1. Criar `src/lib/lms/trilha-7.ts` seguindo o padrão de `LmsTrilha`/`LmsLesson`/`QuizQuestion` (ver `trilha-1.ts`), com os 13 módulos e 5 questões de quiz por módulo.
2. Registrar `trilha7` em `src/lib/lms/index.ts` (import, export, `LMS_TRILHAS`).
3. Gerar migration SQL (`supabase/migrations/20260701230000_lms_seed_trilha_perfumaria.sql`) inserindo em `trilhas` (is_global=true, ordem=11), `modulos`, `aulas` (tipo_conteudo='texto'), `provas` e `questoes_prova` — espelhando o conteúdo do passo 1.
4. Aplicar a migration no projeto Supabase (`zsczxblhtdhpdqvkpuwz`) via MCP `apply_migration`.
5. Verificar contagem de linhas inseridas e rodar `npx tsc --noEmit` + `npm run build`.

## Notas
- Conteúdo fornecido integralmente pelo usuário no chat (13 módulos de treinamento em perfumaria).
- `publico_alvo` = NULL (trilha geral, como as trilhas 1-4 core).
- xp_reward = 100/módulo, aprovacao_minima = 70, seguindo o padrão das trilhas existentes.
